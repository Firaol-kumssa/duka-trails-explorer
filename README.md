# DUKA Hiking & Travel Website

A modern, responsive travel and tourism website built for **DUKA Hiking & Travel**, showcasing Ethiopia's breathtaking hiking destinations, guided tours, travel news, and adventure experiences.

## 🌍 Live Website

https://duka-trails-explorer.firaol-kumssa.workers.dev

---

## 📖 About

DUKA Hiking & Travel is an Ethiopian travel company dedicated to introducing travelers to the country's most spectacular landscapes, cultural heritage, and outdoor adventures.

This website provides an engaging digital experience where visitors can discover destinations, browse tours, stay updated with company news, and contact the team to plan unforgettable journeys across Ethiopia.

---

## ✨ Features

- Modern responsive design
- Beautiful landing page with animated hero section
- Featured Ethiopian destinations
- Guided hiking and tour packages
- News & announcements section
- Interactive gallery
- Contact form
- WhatsApp and Telegram integration
- Social media links
- Mobile-friendly navigation
- Fast loading performance
- SEO-friendly structure
